SVG — نظري
  • ما هي أكواد SVG؟ — Scalable Vector Graphics
  • مزايا SVG: التوسع، خفة الوزن، التعديل
  • مكونات كود SVG والجذر الأساسي <svg>
  • الأشكال: rect, circle, ellipse, line, path
  • خصائص المستطيل: width, height, x, y, rx, ry
  • استخدام النصوص <text> واتجاه RTL
SVG في Power BI
  • استخدام SVG في المخططات الأفقية

DAX + SVG
  • إنشاء مقياس SVG داخل Power BI
  • إضافة data:image/svg+xml;utf8 للكود
  • تحويل نوع البيانات إلى Image URL
  • إنشاء Progress Bar ديناميكي
